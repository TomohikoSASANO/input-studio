# Input Studio (desktop)

既存PDFの上に「タグ（入力欄）」を配置し、入力値を重ねて完成PDFを出力するデスクトップアプリです。

## 開発

- **バックエンド**: `app.py`（Python / pywebview）
- **フロント**: `ui/`（HTML/CSS/JS）

## 実行（開発環境）

```bash
python app.py
```

## ビルド（PyInstaller）

```bash
python -m PyInstaller InputStudio.spec
```

## メモ

- OCRは扱いません（フォーム付きPDF前提でもありません）
- プレビューはPNGを生成して表示します
